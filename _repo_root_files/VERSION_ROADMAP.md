# Version Strategy & Roadmap

## Semantic Versioning
MAJOR – Breaking changes
MINOR – New features
PATCH – Fixes/optimizations

## Roadmap (suggested)
- v1.1.0: richer charging phase analytics + trend defaults
- v1.2.0: optional Victron MQTT path, multi-battery abstraction
- v2.0.0: unified diagnostics layer + cross-package health dashboard
